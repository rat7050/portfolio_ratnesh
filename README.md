# Ratnesh Kumar Portfolio Website

Personal portfolio website for Ratnesh Kumar, built as a responsive static site with HTML, CSS, and JavaScript.

Live preview: [https://ratnesh-portfolio-20260413.vercel.app](https://ratnesh-portfolio-20260413.vercel.app)

## Overview

This website showcases:

- profile and introduction
- featured GitHub projects
- about page with skills and certifications
- contact links for GitHub, LinkedIn, and email
- GitHub profile picture integration

## Project Structure

```text
.
|-- index.html
|-- projects.html
|-- about.html
|-- contact.html
|-- styles.css
|-- script.js
`-- README.md
```

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Vercel for deployment

## Pages

- `index.html` - landing page and featured work
- `projects.html` - selected GitHub projects
- `about.html` - profile, strengths, and certificate section
- `contact.html` - direct contact and collaboration details

## Run Locally

This is a static website, so no build step is required.

1. Download or clone the project.
2. Open `index.html` in your browser.

## Customize Content

Most portfolio content is managed from `script.js`.

You can update:

- name and role
- bio text
- GitHub and LinkedIn links
- profile image
- skills and strengths
- project cards
- certifications section

## Deployment

This project is deployed on Vercel.

To deploy again with Vercel CLI:

```bash
vercel deploy --prod
```
